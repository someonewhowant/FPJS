# Модуль 4: Функции высшего порядка (HOF)

Функция высшего порядка (Higher-Order Function) — это функция, которая делает хотя бы одно из двух:
1. Принимает одну или несколько функций в качестве аргументов.
2. Возвращает другую функцию в качестве результата.

## 1. Принятие функций в качестве аргументов

Это позволяет нам абстрагировать *действие*, а не только данные.

**Пример (самодельный фильтр):**
```javascript
const filter = (list, predicate) => {
  const result = [];
  for (const item of list) {
    if (predicate(item)) {
      result.push(item);
    }
  }
  return result;
};

const isAdult = user => user.age >= 18;
const users = [{ name: 'Ivan', age: 15 }, { name: 'Maria', age: 20 }];

const adults = filter(users, isAdult);
```
*Пошаговое объяснение:*
1. Мы создаем универсальную функцию `filter`, которая не знает, по какому правилу нужно фильтровать.
2. Она принимает `predicate` — функцию, которая возвращает `true` или `false`.
3. Мы передаем `isAdult` внутрь `filter`. Теперь `filter` использует логику `isAdult` для проверки каждого элемента.

## 2. Возврат функций из других функций

Это позволяет нам создавать "фабрики функций" или специализированные версии функций.

**Пример (создатель множителей):**
```javascript
const createMultiplier = (multiplier) => {
  return (value) => value * multiplier;
};

const double = createMultiplier(2);
const triple = createMultiplier(3);

console.log(double(5)); // 10
console.log(triple(5)); // 15
```
*Пошаговое объяснение:*
1. `createMultiplier` принимает число `multiplier` и возвращает новую функцию.
2. Возвращаемая функция "запоминает" значение `multiplier` благодаря замыканию.
3. Мы создаем `double`, вызывая `createMultiplier(2)`. Теперь `double` — это функция, которая всегда умножает на 2.

## 3. Зачем это нужно?
- **Абстракция**: Мы можем выделить общие паттерны (обход массива, обработка ошибок) и передавать в них специфическую логику.
- **Композиция**: Мы можем строить сложные функции из простых маленьких кусочков.
- **Декларативность**: Мы описываем задачу на более высоком уровне.
