# Модуль 2: Чистые функции (Pure Functions)

Чистая функция — это функция, которая:
1. **Детерминирована**: Один вход — один и тот же выход.
2. **Без побочных эффектов**: Не меняет ничего снаружи.

## 1. Ссылочная прозрачность
Вызов функции можно заменить её результатом без изменения поведения программы.
