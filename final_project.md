# Финальный проект: Mini-Ramda

Ваша задача — реализовать собственную функциональную библиотеку по образу и подобию **Ramda.js**. Это поможет вам закрепить все изученные темы: от каррирования до глубокой рекурсии.

## 1. Основные принципы библиотеки

1. **Auto-Currying**: Все функции в библиотеке должны быть автоматически каррированы. Если функция ожидает 3 аргумента, а передано 2, она должна вернуть новую функцию.
2. **Data Last**: Данные, над которыми производится действие (например, массив или объект), всегда передаются последним аргументом. Это облегчает композицию.
3. **Иммутабельность**: Ни одна функция не должна мутировать входные данные.
4. **Pure Functions**: Все функции должны быть чистыми.

## 2. Список функций для реализации

### Ядро и Композиция
- `curry(fn)`: Делает функцию каррированной.
- `pipe(...fns)`: Композиция слева направо.
- `compose(...fns)`: Композиция справа налево.

### Работа со списками (List)
- `map(fn, list)`: Каррированный map.
- `filter(predicate, list)`: Каррированный filter.
- `reduce(fn, init, list)`: Каррированный reduce.
- `find(predicate, list)`: Поиск первого элемента.
- `pluck(key, list)`: Выбирает значение по ключу из каждого объекта в списке.

### Работа с объектами (Object)
- `prop(key, obj)`: Чтение свойства.
- `path(keys, obj)`: Чтение глубоко вложенного свойства по массиву ключей (через рекурсию).
- `assoc(key, value, obj)`: Иммутабельное обновление одного свойства.
- `pick(keys, obj)`: Создание нового объекта только с указанными ключами.

### Логика (Logic)
- `equals(a, b)`: Глубокое сравнение (рекурсивное).
- `ifElse(cond, onTrue, onFalse)`: Функциональный ветвитель.
- `allPass(predicates)`: Проверяет, что все условия верны.

## 3. Этапы выполнения
1. Напишите тесты для каждой функции (используйте `test.js` в корне).
2. Реализуйте `curry` и `pipe` — это фундамент.
3. Перейдите к простым методам списков.
4. Реализуйте рекурсивные функции (`path`, `equals`).

*Подсказка: Для реализации `curry` используйте `fn.length` для определения количества ожидаемых аргументов.*
