# Реализация Mini-Ramda

## Описание шагов

### 1. Ядро (Core)
- **Curry**: Позволяет функциям принимать аргументы по частям. Если аргументов меньше чем `fn.length`, возвращается новая функция.
- **Pipe/Compose**: Инструменты для композиции функций.

### 2. Списки (List)
- **map/filter/reduce**: Каррированные версии стандартных методов. Работают по принципу "Data Last".
- **head/tail**: Утилиты для деструктуризации списков.

### 3. Объекты (Object)
- **prop/path**: Безопасное чтение данных.
- **assoc**: Иммутабельное обновление (создание копии через spread).
- **pick**: Фильтрация ключей объекта.

### 4. Логика (Logic)
- **equals**: Глубокое рекурсивное сравнение структур.
- **allPass**: Комбинирование предикатов.
