# 🧩 Функциональное программирование на JavaScript (Core + Patterns)

Добро пожаловать в интенсивный курс по функциональной парадигме в JavaScript. Этот репозиторий содержит структурированные материалы для перехода от императивного "пошагового" стиля к декларативному, чистому и предсказуемому коду.

---

## 🚀 О курсе

Курс построен по принципу "от простого к сложному". Мы начнем с изменения мышления и закончим созданием собственной библиотеки функциональных утилит (клона Ramda.js).

### Как обучаться:
1.  **Теория**: Изучите модуль в формате Markdown.
2.  **Проверка**: Пройдите квиз в папке `quiz/` для закрепления материала.
3.  **Практика**: Изучите исходный код финального проекта, чтобы увидеть, как теория применяется в реальных инструментах.

## 🏆 Финальный проект: Mini-Ramda

В конце курса вы изучите реализацию **Mini-Ramda** — полнофункционального клона популярной библиотеки Ramda.js.

*   **Локация кода**: `FPJS/src/`
*   **Описание реализации**: [IMPLEMENTATION.md](IMPLEMENTATION.md)
*   **Тесты**: Проверьте работоспособность командой `node FPJS/test.js`

### Ключевые особенности проекта:
- Автоматическое каррирование всех функций.
- Принцип "Data Last" для удобной композиции.
- Полная иммутабельность.
- Глубокое сравнение структур данных.
